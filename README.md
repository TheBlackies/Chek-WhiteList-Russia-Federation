# Скрипт на проверку активность Белых Списков РФ


*Данный скрипт проверяет активны ли в вашей области белые списки или нет.*

---

### Для использования скрипта нужны:

|Python|
|---|
|Библиотека ping3 для Python|

Команда для установки библиотеки: pip install ping3
